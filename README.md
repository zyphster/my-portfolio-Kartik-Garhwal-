# My Portfolio Website

This portfolio contains my landing pages, websites, ad creatives and digital work.

## Structure

- index.html = Main portfolio homepage
- style.css = Main styling
- script.js = Tabs filter functionality
- landing-pages/ = Landing page demos
- websites/ = Website demos
- assets/images/ = Preview images and ad creatives

## How to add new landing page

1. Create folder:

landing-pages/design-2/

2. Add index.html inside it.

3. Add preview image in:

assets/images/design-2-preview.jpg

4. Add new card in main index.html.
